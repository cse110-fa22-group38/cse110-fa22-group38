This page is the result of team project made for CSE110 - Software Engineering at the University of California, San Diego during Fall 2022.

Our project is a planner specifically designed for students in schools of quarter system. The planner consists today view, week view and quarter view. It also has a login page, sign up page and settings page, allowing users to create an account and customize settings.

For this planner, you are able to add events so that it can show on the calendar. The event can be a homework, a exam or a task. You can set its location and the time. Each event would show in the views differently.
