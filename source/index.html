This page is the result of team project made for CSE110 - Software Engineering at the University of California, San Diego during Fall 2022 by group 38.

Our project is a planner specifically designed for students in schools using a quarter system, such as UCSD. The planner consists 3 primary views: daily view, weekly view and quarter view. It also has a login page, sign up page and settings page, allowing users to create an account and customize settings.

For this planner, you are able to add events that show up in each view. The event can be a task (such as an assignment), an event, or an exam. You can set their location, time, and even group events together by giving them the same relation. You can click on an event to see more details.
